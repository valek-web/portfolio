Навыки

Что я использую в работе

HTML5

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Jquery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует

React

Эта библиотека позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваши цели

Git

Git — это система, которая позволяет сразу нескольким разработчикам сохранять и отслеживать изменения в файлах вашего проекта

Gulp

Gulp - это инструмент для автоматизации рутинных задач, которые возникают при веб-разработке.

Redux/ Redux toolkit

Redux — библиотека управления состоянием приложений, написанных на JavaScript. Она помогает писать приложения, которые ведут себя стабильно/предсказуемо

Sass

SASS – это препроцессор CSS, основной задачей которого является расширение возможностей написания CSS кода

БЕМ

Методология БЭМ — это набор универсальных правил, которые можно применять независимо от используемых технологий

Material-ui

Material-ui — reactJS's фрэймворк, предоставляющий готовые google решения для быстрой и довольно простой web разработки

Styled Components

Styled Components — одно из популярных решений написания кода методом CSS in JS.

TypeScript

TypeScript — это язык программирования построенный на базе JavaScript, в котором исправлены многие его недостатки.

MobX

MobX — это автономная библиотека, для управления фронтенд-состоянием приложения.
